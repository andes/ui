PlexList

Selector: plex-list

Exported as: PlexListComponent
Propiedades
Nombre Descripción
@Input()

striped: boolean

Renderiza el listado franjeado (items con fondo gris intercalados).
@Input()

height: string

Valor de la altura del listado (necesario para el funcionamiento del scroll infinito).
@Output()

scrolled: EventEmitter<void>

Evento emitido para recuperar más items.
@Input()

selected: boolean

Establece si el item ha sido seleccionado. *aplica al plex-item
