Componentes

plex-detail

  • Componente que muestra el detalle de un item.
  • Se emplea para visualizar datos de perfil de individuos (pacientes, miembros de equipos médicos, administrativos) o bien de elementos genéricos (registros, conceptos, etc).
  • El componente presenta dos atributos: uno de direccionalidad y otro de tamaño: info y detail
