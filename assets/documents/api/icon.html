Plex Icon

    <plex-icon prefix="adi" name="load"></plex-icon>

Componente

PlexIcon

Selector: plex-icon

Exported as: PlexIconComponent
Propiedades
Nombre Descripción
@Input()

name: string

Nombre del icono tanto de Material Design Icons (MDI) como de Andes Font (ADI)
@Input()

prefix: string

Establece el prefijo del sistema de iconos a utilizar (MDI o ADI)
@Input()

type: string

Especifica la tipología, y por ende, el color que adquiere el componente. Los valores: 'success' | 'info' | 'warning' | 'danger' | 'primary'.
@Input()

size: string

Los valores que se pueden asignar son: '18' | '24' | '36' | '48'.
