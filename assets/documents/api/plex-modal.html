Componente

PlexModal

Selector: plex-modal

Exported as: PlexModalComponent
Propiedades
Nombre Descripción
@Input()

size: 'sm' | 'md' | 'lg' = 'md'

Especifica el tamaño de la ventana del componente, respecto a la ventana del navegador.
sm: 25%; md: 50%; lg: 80% del área visible de la ventana.
@Input()

allowClose: boolean = false

Muestra una cruz para cerrar el modal.
@Input()

allowBackdropClose: boolean = true

Habilita cerrar el modal haciendo haciendo click afuera.
@Input()

allowEscClose: boolean = true

Habilita cerrar el modal con la tecla [ESC].
@Input()

startOpen: boolean = false

El modal se visualiza desde el inicio.
@Output()

closed: EventEmitter

Emite un evento cuando se cierra el modal.
public method

show()

Muestra el modal, que por defecto está oculto
public method

close()

Cierra el modal y emite el evento closed

public method

backdropClick()

Cierra el modal al hacer click afuera del mismo, si allowBackdropClose = true
