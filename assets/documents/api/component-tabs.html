Tabs

Directives

PlexTabs

Selector: plex-tabs

Exported as: PlexTabsComponent
Propiedades
Nombre Descripción
@Input()

label: string

Muestra el titulo de la pestaña.
@Input()

icon: string

Renderiza un icono de (Material Design Icons / Andes Font) ingresando sólo el nombre del icono (sin prefijos).
@Input()

active: boolean

Establece si la pestaña se encuentra en estado activo. *aplica al plex-tab
@Input()

allowClose: boolean

Define si se permite cerrar la pestaña o no. Útil en el caso de las pestañas dinámicas.
@Input()

color: string

Define el color del texto del titulo.
