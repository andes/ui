Plex Button

    <plex-button label="Success" type="success" icon="account"></plex-button>

Componente

PlexButton

Selector: plex-button

Exported as: PlexButtonComponent
Propiedades
Nombre Descripción
@Input()

label: string

Etiqueta para mostrar en el botón.
@Input()

icon: string

Icono del botón.
@Input()

type: string

Color del botón. Valores: 'success' | 'primary' | 'danger' | 'info' | 'warnings'

@Input()

size: string

opcional

Establece el tamaño del botón. Valores 'sm' | 'lg' .
@Output()

click: EventEmitter<bool>

Ocurre cuando se hace click en el botón.
