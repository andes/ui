plex-modal, ventanas modales

Las ventanas modales plex-modal pueden cumplir varias funciones:
  • Realizar acción contextual de un módulo
  • Advertir al usuario de tomar una acción necesaria para continuar
  • Simplemente comunicar, a manera de ventana de información o recodatorio
  • Se activa un scroll vertial interno en caso de contenido extenso

plex-modal normal, con size por defecto

1. Modal alerta invasivo, bloquea el flujo de la aplicación.

plex-modal: size="md"

2. Modal 50% ancho con texto largo.

plex-modal size="sm"

3. Modal 25% ancho, se puede usar como confirm.

plex-modal: size="lg"

4. Modal 80% ancho, sin ícono.
