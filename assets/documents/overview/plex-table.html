Componentes

plex-table

  • Renderiza una tabla HTML mediante atributos y directivas.
  • Se utiliza para proyectar grandes volúmenes de datos, preferentemente textual y no interactiva.
  • Se emplea en contextos en los que el usuario necesita reordenar filas y columnas.
  • Generalmente acompaña al componente plex-wrapper en la tipología de pantallas listado-detalle.

  • La tabla se construye a partir de las siguientes directivas:
  • columns directiva y display-column
